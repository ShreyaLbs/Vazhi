# 🌿 Vazhi (വഴി)
**Your Path to Government Schemes in Kerala.**

Vazhi is a highly accessible, people-first AI assistant designed to help every citizen discover the government schemes they deserve — in plain, simple language. Built with an inclusive UI targeting non-technical users, it completely removes the friction between citizens and available government aid.

## ✨ Key Differentiating Features
- **Bilingual Support:** Seamless, instant English & Malayalam (മലയാളം) toggle right in the navigation bar.
- **The "Timeline Path" Chat UI:** A unique, beautiful chat interface mapping the user's journey along a visual timeline, replacing generic chat bubbles.
- **People-First Accessibility:** Features massive, highly legible typography (base 18px) designed specifically for readability on budget Android devices.
- **Tap-First Interaction:** A highly visible grid of massive emoji topics (🎓 Education, 🏥 Health, 🏠 Housing, etc.) eliminates the need for typing out queries.
- **Instant Document Checklists:** Automatically extracts and beautifully renders the explicit documents required (e.g., Aadhaar, Ration Card) as an interactive checklist before applying, saving users from multiple trips to government offices.
- **1-Tap WhatsApp Sharing:** Natively integrated WhatsApp sharing buttons strictly formatted for easy forwarding to family members or Akshaya centers.

## 🛠️ Technology Stack
- **Backend:** Python (Flask)
- **Frontend:** Responsive vanilla HTML/CSS/JS (Zero framework overhead)
- **Aesthetics:** Premium Glassmorphism UI, fluid animated CSS gradients, and modernized Google Typography (Outfit & Plus Jakarta Sans).

## 🚀 Deployment
This app is completely production-ready with configured `requirements.txt` and `Procfile`. It is designed to be deployed directly to cloud services like [Render.com](https://render.com).
